== Description ==

WordThree integrates with your WordPress website to easily allow users to login or register using their MetaMask wallet. Once activated, a MetaMask login button is automatically 
added to the default WordPress login page. Alternatively, you can use one of the following shortcodes to place a MetaMask button on your website. The plugin includes a couple of ways
to present the MetaMask button, either blue, white, large and compact.

== Shortcodes ==

[wordthree_metamask_login]
[wordthree_metamask_register]
[wordthree_metamask_link]

If you have any questions, please email support@wordthree.co.uk

== Pre-requirements ==

1. Once you have chrome browser open, head over to MetaMask' website and click on the download link. After installing it will prompt you 
to create an account: https://metamask.io/download.html
